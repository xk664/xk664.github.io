# 注意事项
1.dp的遍历取决于状态转移方程，计算某个状态的前提是其前置状态已经计算完毕
```java 
	dp[i][j]=dp[i+1][j-1]+dp[i-1][j-1]
	//依赖了前一列的计算结果，因此要先遍历j
```

2.DFS和BFS中用visited标记的意义就是防止无限搜索下去
3.判断一个点是否在斜率为k的直线上，可用向量交叉相乘相等，来避免直接用处罚造成的精度问题
4.进制转换
	**从最高位开始，每一步将当前累计结果乘以进制基数，再加上当前位的数字**，重复此过程直到处理完所有位。
5.能用整型尽量用，因为用double，会造成精度丢失，比如MAth.sqrt()-->大概值
6.滑动窗口中，队列存的是数组下标（用来判断当前最大值是否再当前窗口内）
7.用包装类比较的时候用equals包括Integer
8.头插法和三指针均可以实现链表反转

## Floyed

- 变种问题
1.判断任意两点是否可达
2.求任意两点的最短距离

==注意是否初始化为无穷大或者无穷小的问题==

1、确定dp数组（dp table）以及下标的含义

```java
grid[i][j][k] = m，//表示 节点i 到 节点j 以[1...k] 集合中的一个节点为中间节点的最短距离为m
```

2.状态方程
	我们分两种情况：

1. 节点i 到 节点j 的最短路径经过节点k
2. 节点i 到 节点j 的最短路径不经过节点k

对于第一种情况，`grid[i][j][k] = grid[i][k][k - 1] + grid[k][j][k - 1]`

节点i 到 节点k 的最短距离 是不经过节点k，中间节点集合为[1...k-1]，所以 表示为`grid[i][k][k - 1]`

节点k 到 节点j 的最短距离 也是不经过节点k，中间节点集合为[1...k-1]，所以表示为 `grid[k][j][k - 1]`

第二种情况，`grid[i][j][k] = grid[i][j][k - 1]`

如果节点i 到 节点j的最短距离 不经过节点k，那么 中间节点集合[1...k-1]，表示为 `grid[i][j][k - 1]`

因为我们是求最短路，对于这两种情况自然是取最小值。

即： `grid[i][j][k] = min(grid[i][k][k - 1] + grid[k][j][k - 1]， grid[i][j][k - 1])`

3.初始化
初始给出的数据，中间没有任何，节点，所以直接k\==0,dp\[i]\[j]\[0]=w;
几何来看，初始化的是第一层

4.遍历顺序

由于初始化的第一层，我们只需要一层一层遍历上去，所以最外层循环为k，里面的i和j的顺序无所谓

5.输出
dp\[i]\[j]\[n]即为i到j的最短距离

- ==空间优化==

有状态方程可以知道，当前状态只和当前层和前一层有关，于是可以由滚动数组优化为dp\[i]\[j]\[0]和dp\[i]\[j]\[1]两层

继续，由于当前层的dp\[i]\[j]\[k]一定小于等于前一层的dp\[i]\[j]\[k-1]（由状态方程可知）,所以直接可以就一层即可，可以直接去掉z坐标，变成二维数组，初始化时，也直接初始化再当前层

- 判断是否存在负环
如果dist\[i]\[i]<0则说明存在负环

## bellam_ford

作用：解决单源最短路，权值中存在负数的情况

思路：将所有边松弛一遍，可以得到起点到与他相邻一条边的节点的最短距离
	将所有边松弛二遍，可以得到起点到与他相邻两条边（包括一条边）的节点的最短距离
所以我们松弛节点数-1遍即可，因为所有点和起点最多相距n-1条边

- 朴素
储存所有边-->将所有边松弛n-1遍，更新minDist数组
```java
import java.util.*;

public class Main{

    public static void main(String[] args){

        Scanner sc=new Scanner(System.in);

        int n=sc.nextInt();

        int m=sc.nextInt();

        List<int[]> list=new ArrayList<>();

        for(int i=1;i<=m;i++){

            int a=sc.nextInt();

            int b=sc.nextInt();

            int w=sc.nextInt();

            list.add(new int[]{a,b,w});

        }

        int INF=Integer.MAX_VALUE/2;

        int[] minDist=new int[n+1];

        Arrays.fill(minDist,INF);

        minDist[1]=0;

        for(int i=1;i <n;i++){

            for(int[] t:list){

                int st=t[0];

                int ed=t[1];

                int w=t[2];

                if(minDist[st]!=INF&&minDist[ed]>minDist[st]+w){

                    minDist[ed]=minDist[st]+w;

                }

            }

        }

        if(minDist[n]==INF){

            System.out.println("unconnected");

        }else {

            System.out.println(minDist[n]);

        }

    }

}
```

- 优化（SPFA）

实际上只有上次松弛过的顶点才会用来松弛后续的顶点（队列实现）

**步骤：

起点入队
while(队列不为空){
队头元素=poll();
visited=flase(标识这个点不在队列)

松弛他的邻接点{
松弛(更新minDist数组)
如果已经再队列已经有了就不入队，不在则入队(只有进行了松弛才到这一步)
}
}
```java
import java.util.*;

  

public class Main {

    public static void main(String[] args) {

        Scanner sc = new Scanner(System.in);

        int n = sc.nextInt();

        int m = sc.nextInt();

        boolean[] book = new boolean[n + 1];

        List<List<int[]>> charm = new ArrayList<>();

        charm.add(new ArrayList<>());

        int[] cnt=new int[n+1];

        for (int i = 1; i <= n; i++) {

            charm.add(new ArrayList<>());

        }

        for (int i = 1; i <= m; i++) {

            int a = sc.nextInt();

            int b = sc.nextInt();

            int w = sc.nextInt();

            charm.get(a).add(new int[] {b, w});

        }

        int INF = Integer.MAX_VALUE / 2;

        int[] minDist = new int[n + 1];

        Arrays.fill(minDist, INF);

        minDist[1] = 0;

        ArrayDeque<Integer> Q = new ArrayDeque<>();

        Q.offer(1);

        cnt[1]=1;

        book[1] = true;

        boolean flag=false;

        while (!Q.isEmpty()) {

            int t = Q.poll();

            book[t] = false;

            for (int[] num : charm.get(t)) {

                if (minDist[num[0]] > minDist[t] + num[1]) {

                    minDist[num[0]] = minDist[t] + num[1];

                    if (!book[num[0]]) {

                    Q.offer(num[0]);

                    cnt[num[0]]++;

                    book[num[0]] = true;

                    if(cnt[num[0]]>=n){

                        flag=true;

                        while(!Q.isEmpty())Q.pop();//清空队列

                        break;

                    }

                }

                }

            }

        }

        if(flag==true){

            System.out.println("circle");

        }

        else if (minDist[n] == INF) {

            System.out.println("unconnected");

        }

        else {

            System.out.println(minDist[n]);

        }

    }

}
```

## 解决思路

假设你有了前面的状态，然后怎么可以得到当前状态，比如对于当前物品选或者不选，然后列出状态转移方程，最后初始化（哪些递推的底层，也就是不满足状态转移方程的状态，比如当前状态dp\[i]依赖dp\[i-1],那么dp\[0],就不满足，因为他前面已经没有状态了，所以就要把他初始化）
